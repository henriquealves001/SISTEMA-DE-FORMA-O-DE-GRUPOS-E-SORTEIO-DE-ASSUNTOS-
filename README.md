# Sorteador de Grupos

Projeto desenvolvido em **Java** com o objetivo de auxiliar professores ou usuários em geral na organização de **grupos de alunos** e no **sorteio de temas/assuntos** para cada grupo.

## Objetivo

O programa foi criado para facilitar a divisão de alunos em grupos de forma prática, rápida e aleatória, evitando organização manual e tornando o processo mais eficiente em sala de aula.

Além do sorteio dos grupos, o sistema também permite sortear assuntos para cada grupo, exibindo o resultado final de forma organizada.

## Funcionalidades

- Cadastrar alunos
- Cadastrar assuntos
- Listar alunos cadastrados
- Listar assuntos cadastrados
- Sortear grupos aleatoriamente
- Sortear assuntos para os grupos
- Exibir resultado final
- Refazer sorteio quando necessário
- Menu interativo no terminal
